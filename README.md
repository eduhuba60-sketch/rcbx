# <div align='center'>Gifted Baileys</div>

Gifted Baileys is a WebSockets-based TypeScript/Javascript library for interacting with the WhatsApp Web API.

> [!TIP]
> If you need to send buttons without using official **[baileys](https://npmjs.com/package/baileys)** please check out **[gifted-buttons](https://npmjs.com/package/gifted-btns) package.**

> You might still want to use **[gifted-baileys@2.0.0](https://www.npmjs.com/package/gifted-baileys/v/2.0.0)** which has lts as it was fixed before official baileys was fixed.
